The „quellcode“ folder contains 7 parts. 

1. bench —— calculate the BSDS benchmark scores of image segmentations

2. edison_matlab_interface —— matlab wrapper for mean shift clustering

3. Mean_shift_horse_alternative —— A alternative trial for mean shift clustering under    
                                   Weizmann horse dataset

4. matconvnet-new-solvers ——- a toolbox for constructing CNN    

5. Training_BSDS500 —— code for training network under BSDS 500 dataset

6. Training_horse  ——  code for training network under Weizmann horse dataset

7. Visualization_PCA_features ——— code for visualization of PCA-transformed features
